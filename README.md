# pot-app-translate-plugin-glm

GLM 翻译插件 - 一个为pot-app开发的翻译插件

## 简介
这是一个基于 GLM API 的翻译插件，支持多语言翻译，提供高质量的翻译结果。

## 前置要求
- [pot-app](https://github.com/pot-app/pot-app) - 请先下载并安装 pot-app
- [GLM API开放平台](https://bigmodel.cn)- 需要在 GLM 官网申请 API 密钥

## 功能特点
- 支持模型选择：GLM-4.7等
- 支持多种语言之间的互译
- 提供专业、流畅的翻译结果

## 安装方法
1. 在pot-app中点击"偏好设置"
2. 选择"服务设置"
3. 点击"翻译-添加外部插件"
4. 选择本插件的发布包（右侧“Releases”）进行安装

## 配置说明
1. 选择翻译模型
2. 输入你的 GLM API 密钥

## 使用说明
安装并配置完成后，在 pot-app 的翻译引擎列表中选择 GLM 即可使用。

## 注意事项
- 请妥善保管你的 API 密钥
- 翻译服务需要联网使用
